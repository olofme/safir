do not have the libiomp5md.DLL file.
Attached you can find 3 versions of this dll.

1. Libiomp5md.dll.in_IA32
Windows 7, was located in 
C:\Program Files(x86)\Intel\ComposerXE-2011\redist\ia32\compiler   
Delete ".in_IA32" from the file name.

2. Libiomp5md.dll.in_intel64
Windows 7, was located in
C:\Program Files(x86)\Intel\ComposerXE-2011\redist\intel64\compiler
Delete ".in_intel64" from the file name.

3. Libiomp5md.zip
was located in
